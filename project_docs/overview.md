# Project Overview: Sport Journal Web App

## High-level Background
Sport Journal Web App helps users easily log, track, and analyze their fitness workouts, with a focus on simplicity and quick access to historical data.

## Core Vision
- Provide a convenient way for users to keep a training journal (workouts, exercises, sets, notes, body measurements, moods)
- Allow simple editing, searching, and viewing past workouts
- Suitable for both casual and motivated trainees

## Main Objectives
- Fast workout logging experience
- Powerful browsing and editing of workout history
- Support for custom notes, mood tracking, and simple metrics

## Problems Solved
- Forgetting what was done last session
- No need for paper journals
- Can review, edit and analyze past training
